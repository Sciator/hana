## Building and dev

### prerequisites
   - [node](https://nodejs.org/) installed
   - [yarn](https://yarnpkg.com/) installed

### distribution
   - ```yarn dist```

### dev
   - ```yarn install``` (if not called already)
   - ```yarn dev```
